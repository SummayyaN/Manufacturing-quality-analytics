📊 Manufacturing Quality Analytics using SQL & Power BI
📌 Project Overview

This project demonstrates how data analytics can be applied in a manufacturing environment to monitor quality performance, identify defect drivers, and support data-driven decision-making.
The analysis focuses on inspection data, supplier quality, and production operations using SQL and Power BI.

The project is designed to simulate a real-world manufacturing quality scenario, suitable for Data Analyst roles in IT and analytics-driven organizations.

🎯 Business Problem

Manufacturing organizations face challenges such as:

High rejection and rework rates

Lack of visibility into defect patterns

Inconsistent supplier quality

Difficulty tracking quality KPIs across machines and shifts

This project aims to solve these problems by transforming raw manufacturing data into actionable insights.

🛠 Tools & Technologies Used

SQL – Data aggregation, KPI calculations, analysis

Power BI – Interactive dashboards & visual storytelling

Excel / CSV – Data preparation and validation

📂 Datasets Used

The project uses simulated but realistic manufacturing data:

Inspection Data

Batch-wise production and rejection details

Defect types, machines, shifts, product types

Supplier Data

Supplier-wise material received and rejected quantities

Production Data

Planned vs actual production

Machine downtime data

Each dataset contains 100 records to closely represent real operational data.

🔑 Key KPIs & Metrics

Total Production

Total Rejection

Rejection Percentage

First Pass Yield (FPY)

Defect-wise Rejection Contribution (Pareto)

Machine-wise and Shift-wise Quality Performance

Supplier Rejection Percentage

Downtime vs Quality Impact

**📊 Dashboard Structure (Power BI)**
🔵 Page 1: Quality Overview (Management View)

KPI cards: Total Production, Total Rejection, Rejection %, FPY

Defect trend over time

Machine-wise rejection comparison

🔵 Page 2: Defect & Process Analysis (Quality Team View)

Pareto chart (Defect Types – 80/20 rule)

Shift-wise defect analysis

Batch-level inspection details

🔵 Page 3: Supplier & Operations Analysis (SUpplier & Operational View)

Supplier rejection percentage scorecard

Downtime vs rejection correlation analysis

📈 Key Insights Generated

Identified top defect types contributing to the majority of rejections using Pareto analysis

Highlighted machines and shifts with higher rejection rates

Evaluated supplier quality performance to support vendor improvement actions

Analyzed the relationship between downtime and quality loss
